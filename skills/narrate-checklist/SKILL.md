# Checklist Narrative - From Checklist to Comprehensive Guide

Converts PDF checklists into comprehensive guidance documents explaining context, usage, and troubleshooting.

## Quick Reference

**Input**: PDF checklist (table format with checkboxes)
**Output**: Comprehensive guide with context, per-item detail, workflow, troubleshooting
**Language**: Formal Bahasa Indonesia
**Key**: Checklist PDF remains as appendix/lampiran; this creates the explanation document

## Output Structure

- **A. Tujuan dan Ruang Lingkup** - Purpose, scope, when to use, who's responsible
- **B. Penjelasan Per Fase/Kategori** - For each phase: overview, timeline, success criteria, item-by-item details
- **C. Workflow Penggunaan Checklist** - How to use (initiation, execution, review, documentation)
- **D. Matriks Penanggung Jawab** - Responsibility matrix
- **E. Troubleshooting dan Escalation** - Common problems, solutions, escalation paths
- **F. Referensi dan Dokumen Terkait** - Related SOPs, forms, systems, contacts
- **Lampiran** - Reference to original checklist PDF
