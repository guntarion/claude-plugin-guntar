# Setup Playwright MCP

Quickly install and configure Playwright Model Context Protocol server for Claude Code, enabling browser automation capabilities.

## What it does
- Installs Playwright MCP server via Claude Code CLI
- Verifies installation and checks browser availability
- Provides next steps for browser-specific setup

## Usage
Use this skill when you need to:
- Set up Playwright MCP for the first time
- Reinstall or update the MCP server
- Check Playwright MCP configuration status

See [reference.md](./reference.md) for detailed documentation and troubleshooting.
