# SOP Content Improvement - Content Quality and Completeness

Reviews and improves SOP documents ensuring completeness, proper structure (A-H sections), and generalization for all PLN Corporate University units.

## Quick Reference

**Input**: Document path(s), output path, document type, missing sections list, merge decisions
**Output**: Improved markdown with complete A-H structure, generalized content
**Language**: Formal Bahasa Indonesia
**Key**: Conservative editing - preserve original meaning, add only for completeness

## Main Tasks

1. **Create missing A-H sections** (Tujuan, Ruang Lingkup, Referensi, Istilah, Perlengkapan, K3, Material, Langkah Kerja)
2. **Merge duplicate documents** intelligently
3. **Generalize unit-specific references** (UPDL Pandaan â†’ Unit PLN Corporate University)
4. **Convert Business Process diagrams** to narrative format

## Conservative Principles

**CAN do**: Add missing sections, clarify wording, generalize unit references, merge logically
**CANNOT do**: Change procedures, remove steps, invent references/terms, modify technical requirements
