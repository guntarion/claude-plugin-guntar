# Setup Context7 MCP

Quickly install and configure Context7 Model Context Protocol server for Claude Code, enabling access to up-to-date library documentation and code examples.

## What it does
- Installs Context7 MCP server via Claude Code CLI with HTTP transport
- Configures authentication with Context7 API key
- Verifies installation and connectivity
- Provides access to comprehensive library documentation

## Usage
Use this skill when you need to:
- Set up Context7 MCP for the first time
- Reinstall or update the MCP server
- Check Context7 MCP configuration status
- Access library documentation through Context7

See [reference.md](./reference.md) for detailed documentation and troubleshooting.
