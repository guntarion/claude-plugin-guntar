# Format Tables in Word Documents

Applies consistent formatting to all tables: centers headers, removes bold from first column body, left-aligns body content.

## Quick Start

```bash
source .venv/bin/activate
python .claude/skills/msword-format-table/msword-format-table.py <input.docx> [output.docx] [--style "Table Grid"]
```
