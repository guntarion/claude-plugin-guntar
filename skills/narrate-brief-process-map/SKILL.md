# Brief Process Map Narrative - Strategic Overview untuk Bab 1

Converts multi-page master process maps into concise strategic overviews (330-550 lines) for Bab 1, focusing on narrative flow without complex diagrams.

## Quick Reference

**Input**: Master process map PDF (multi-page swimlane diagram)
**Output**: 4-layer markdown document (~400 lines, 40-60% shorter than detailed version)
**Target**: Strategic overview for Bab 1 (Gambaran Strategis)
**Language**: Formal Bahasa Indonesia
**Key**: Standalone explanation, B5-friendly formatting, no ASCII flowcharts

## Layer Structure

1. **Ringkasan Eksekutif** (100-150 lines): Strategic overview paragraphs
2. **Peta Navigasi Strategis** (50-100 lines): Nested bullets, simple tables, role guidance
3. **Narrative Detail** (150-250 lines): Per sub-process narrative with brief control points
4. **Referensi** (30-50 lines): Minimal references

**Format**: Nested bullets preferred over wide tables (B5-friendly), flowing paragraphs for narrative
