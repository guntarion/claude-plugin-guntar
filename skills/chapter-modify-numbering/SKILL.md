# Modify Chapter Numbering

Modifies chapter numbering in both filenames and file contents by renaming files and replacing chapter references throughout markdown files.

## Quick Start

```bash
source .venv/bin/activate
python .claude/skills/chapter-modify-numbering/chapter-modify-numbering.py <folder_path> <old_chapter> <new_chapter>
```
