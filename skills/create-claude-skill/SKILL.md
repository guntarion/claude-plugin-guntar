# Create Claude Code Skills

Comprehensive guide for creating well-structured Claude Code skills following official best practices and proven patterns.

## Quick Workflow

1. **Identify skill type:** Script-based or AI-only
2. **Create folder structure:** Self-contained with progressive disclosure
3. **Write concise SKILL.md:** ~50 tokens with links to details
4. **Create reference.md:** Comprehensive documentation with TOC
5. **Add supporting files:** Scripts, templates, examples as needed

## Core Principles

- **Progressive Disclosure:** Load details only when needed
- **Conciseness:** SKILL.md under 50 tokens
- **Self-Contained:** All resources inside skill folder
- **Clear Structure:** Consistent organization across skills
