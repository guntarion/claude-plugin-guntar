# SOP Improvement Orchestration

Orchestration workflow for reviewing and improving SOP documents using parallel sub-agents for content improvement, editing, and structure creation.

## Quick Start

This is an orchestration skill with no script. Follow the workflow in reference.md to:
1. Analyze SOP documents
2. Improve content (parallel agents)
3. Edit and format (parallel batches)
4. Create structure and introductions
5. Validate and report
