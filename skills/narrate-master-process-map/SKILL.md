# Master Process Map Narrative - Strategic Navigation Guide

Converts multi-page master process map swimlane diagrams into comprehensive layered navigation guides for Bab 1.

## Quick Reference

**Input**: Multi-page swimlane diagram (8-15+ pages, strategic framework, Level 1.0 processes)
**Output**: Layered navigation guide (600-1000+ lines) with strategic narrative, navigation tools, detailed narrative, control points
**Language**: Formal Bahasa Indonesia
**Key**: Multi-audience layering, emphasize interconnections, simplify visual complexity

## Layer Structure

1. **Strategic Narrative** - Executive summary with narrative arc
2. **Navigation Tools** - Simplified flowchart/matrix, role-based guidance
3. **Detailed Narrative** - Full narrative per sub-process with control points
4. **Reference** - Link to original swimlane PDF

**Focus**: Strategic positioning, interconnections with upstream (Kantor Pusat) and downstream processes
