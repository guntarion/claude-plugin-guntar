# Compose Preview - Creating Sub-Chapter Introductions

Creates x.x.0 introduction files for book sub-chapters by reading sibling files and synthesizing 3-paragraph overviews in formal Bahasa Indonesia.

## Quick Reference

**Input**: Sub-chapter number, title, source files list, target filename
**Output**: Markdown file with ## heading + 3 paragraphs (context, scope, structure)
**Language**: Formal Bahasa Indonesia
**Key Rules**: Generalize for all PLN Corporate University units, no specific unit names
