# Convert Markdown to Word Document

Converts markdown files to Word format by merging multiple files in alphabetical order with proper B5 JIS formatting and mirror margins.

## Quick Start

```bash
source .venv/bin/activate
python .claude/skills/convert-md-to-docx/convert-md-to-docx.py <folder_or_files> [output.docx]
```
